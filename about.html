LOG VIEWER BETA

Client-side CSV parsing and plotting for ECU tuning logs.

What is it?

A client-side CSV/TXT log viewer for EcuTek style exports. Real-time graphs for RPM, boost, AFR, throttle, ignition, etc. All parsing & rendering happens in your browser. No uploads.

Key features

  • Multi Plot: multiple Y-axes vs Time
  • Mega Plot: compare/offset/scale with point tracking
  • 100% Local: zero server processing
  • Responsive: desktop & mobile
  • Dark UI: matches your theme
  • Session cache: switch pages without re-upload

Supported formats

EcuTek CSV/TXT with a header row and time-series columns (e.g., Time, RPM, Boost, AFR, Throttle, Ignition).

Tech stack

Vanilla JS + HTML5 + CSS3. Plotly.js for charts, PapaParse for CSV. Optimized for large logs—no backend.

Privacy

Logs are processed locally. No network requests for file handling; your data stays on your device.
